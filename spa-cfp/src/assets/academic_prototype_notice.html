Aviso: Protótipo Acadêmico

Este aplicativo (CFP) é um protótipo acadêmico disponibilizado na Internet para testes/uso do desenvolvedor e avaliação dos coordenadores do curso.

Embora exista um esforço para manutenção da segurança dos dados transmitidos para o aplicativo e com a credencial utilizada, não há nenhum compromisso de manutenção dos dados impostados, os quais podem ser apagados a qualquer momento seja por uma alteração na versão do aplicativo ou descontinuidade do mesmo a qualquer tempo.

Para comodidade do usuário o armazenamento local do dispositivo é utilizado para armazenar o token de acesso válido por determinado período.
